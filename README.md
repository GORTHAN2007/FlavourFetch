# FlavourFetch

**Description**: A recipe search tool where users can input what's in their fridge to find meals they can actually make. A better version of a cooking book.

## Core Features

- Filter by ingredient
- View cooking instructions
- "Nutrition" facts toggle

## Public API

- [TheMealDB API](https://www.themealdb.com/api.php)

## Tech Stack

- React
- HTML5
- CSS3
- JavaScript (ES6+)

## React Concepts

- URL parameters for "View Recipe" pages
- Mapping ingredients list

## UI Ideas

- A clean, kitchen-inspired UI with high-quality food photography
